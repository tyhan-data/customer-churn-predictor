# Customer Churn Predictor

A production-ready machine learning project designed to predict customer churn using advanced data science techniques. This repository contains comprehensive data analysis, model development, and a fully functional web application for real-time churn prediction.

## 🎯 Project Overview

Customer churn prediction is a critical business challenge. This project provides:
- **Advanced Predictive Analytics** - Identify customers at risk of leaving
- **Data-Driven Insights** - Understand key factors influencing churn
- **Actionable Intelligence** - Enable proactive retention strategies
- **Interactive Web Application** - Real-time predictions and visualization

## 📊 Live Application

Access the interactive web application here: **[Customer Churn Predictor App](https://customer-churn-predictor-by-tyhan.streamlit.app/)**

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- pip or conda package manager

### Installation

1. **Clone the repository:**
```bash
git clone https://github.com/tyhan-data/customer-churn-predictor.git
cd customer-churn-predictor
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the Streamlit application:**
```bash
streamlit run app.py
```

The application will open in your default web browser at `http://localhost:8501`

### Using Jupyter Notebooks

Launch Jupyter Notebook to explore the analysis:
```bash
jupyter notebook
```
